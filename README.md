#### dependencies
```
mpv
```

#### installation

- from pypi
```bash
pip install ani-cli
```

- from git
```bash
pip install git+https://github.com/sleepntsheep/ani-cli
```

#### usages

```bash
ani 
ani some anime name
```

if you want to select quality

```bash
ani -q
ani -q some anime name 
```
