browse and watch anime (scrape from gogoanime) (wip)

basically ![ani-cli](https://github.com/pystardust/ani-cli) but in python cuz python good

#### demo
![demo](demo.gif)

#### dependencies
```
mpv
```

#### installation

- from pypi
```bash
pip install ani-cli
```

- from git
```bash
pip install git+https://github.com/sleepntsheep/ani-cli
```

#### usages
```bash
ani
```
or
```bash
ani some anime name
```
